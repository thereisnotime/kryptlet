# Security policy

## Supported versions

Only the latest release receives security fixes.

## Reporting a vulnerability

Please report vulnerabilities privately via [GitHub Security Advisories](https://github.com/thereisnotime/kryptlet/security/advisories/new).

Include:
- A description of the vulnerability
- Steps to reproduce
- Impact assessment
- A suggested fix if you have one

Expected response time: 48 hours. Reporters are credited in release notes.

Do not open public issues for security vulnerabilities.
